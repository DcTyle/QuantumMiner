## Working Rule

- Research context explains hypotheses, continuity, archive state, and next-pass framing.
- Engine specifications define runtime behavior, constraints, and canonical implementation rules.
- Physics context contains the theory and reference framing used to motivate or interpret the research.
- Logs record operational output and experiment state transitions.
- Lab notes interpret a specific run and should always be paired with linked CSV artifacts in sequence order.
